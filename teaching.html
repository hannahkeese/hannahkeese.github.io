Lights Out Workshop: this is a worksheet I wrote for Math 2210: Linear algebra, a first course in linear algebra. It explores an application of linear algebra to the electronic 90s game "Lights Out". You can find an online emulator of the game here. The worksheet is too long for a single discussion section, but can be broken up so that some questions are assigned as homework or completed prior to class. Ideally students will have seen determinants and matrix inverses before attempting it.
